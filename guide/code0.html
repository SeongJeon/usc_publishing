Class 네이밍 방법

최종 수정일 : 2017.00.00

1. 공통으로 들어가는 UI를 만들경우 comp-해당UI명 으로 통일한다.
ex: comp-paging , comp-tab...


2. 형태는 같으나 스타일이 다른 경우 언더바(_)로 지정한다.
ex: comp-tab_small, comp-tab_red, comp-tab_type1...

이미지 네이밍 방법

최종 수정일 : 2017.00.00

1. page content에 관련한 이미지의 경우, 사이트 마다의 각 대분류 폴더 안에 저장한다.

2. 공통으로 쓰이는 경우 common 폴더에 저장한다. (해당 폴더안의 스프라이트 이미지 추가를 원할 경우 요청할 것)

3. 임시로 쓰이는 이미지의 경우 사이트 마다의 각 대분류 폴더 안에 temp 폴더 생성 후, 해당 폴더에 저장해서 사용한다.
